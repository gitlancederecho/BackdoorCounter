THE BACKDOOR – Instagram Follower Counter (Digital Smiirl)

How to run:
1) Install Python deps:
     pip3 install flask requests
2) Set IG creds in Terminal:
     export IG_USER_ID=YOUR_IG_USER_ID
     export IG_ACCESS_TOKEN=YOUR_LONG_LIVED_TOKEN
3) Run the backend:
     python3 server.py
4) Open display.html in Chrome/Safari (edit YOUR_HANDLE in the file).
5) Full-screen and mirror your iMac to TVs.

Notes:
• Updates about every 15s (same as physical Smiirl).
• Keep the iMac awake and on stable internet.
